# Установка

Добро пожаловать в проект Kabachok Bot! Эта инструкция поможет вам настроить проект на вашем локальном компьютере.

## Шаги установки

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/omolodec612-spec/kabachok-bot.git
   ```

2. Перейдите в каталог проекта:
   ```bash
   cd kabachok-bot
   ```

3. Установите необходимые зависимости:
   ```bash
   npm install
   ```

4. Запустите проект:
   ```bash
   npm start
   ```

Теперь бот должен быть запущен!
